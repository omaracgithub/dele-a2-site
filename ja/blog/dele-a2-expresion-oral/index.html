DELE A2の口頭表現 — テーマ、例、合格のためのアドバイス

2026年6月更新 · 読了時間10分

DELE A2の口頭試験は12分（準備時間12分を含む）で、2人の試験官の前で行われます。最も緊張する試験ですが、最も予測可能でもあります。テーマは繰り返され、必要な構造は限られています。準備をすれば、あなたの最良の味方にすることができます。

構造：4つのタスク

タスク1 — 準備されたモノローグ（2-3分）

求められること：準備時間の12分間に準備する日常的なテーマについて話します。2つの選択肢が与えられ、そのうちの1つを選びます。

よくあるテーマ：

  • 家族について説明する
  • 仕事や学業について話す
  • 自分の街や地域について説明する
  • 日常のルーチンを話す
  • お気に入りの休暇について話す
  • 自宅やアパートについて説明する

推奨される構造：一般的な導入 → 具体的な詳細3-4点 → 結論または個人的な好み。

例（テーマ：あなたの街）：「私はバルセロナに住んでいます。海岸にある非常に大きな街です。美しいビーチとたくさんの公園があります。私が最も好きなのはゴシック地区で、古い通りがあり、とても良いレストランがあります。また、すべての場所に簡単に行けるので、地下鉄もとても好きです。夏には観光客が多いのが嫌です。」

タスク2 — 写真の説明（2-3分）

求められること：日常的な状況の写真が示されます。見えるものを説明し、試験官の質問に答えます。

典型的な写真：レストランで食事をしている人々、公園にいる家族、市場で買い物をしている人々、ビーチにいる友人のグループ。

説明のための公式：

  • 「写真には...があります。」（人、物）
  • 「彼らは...にいます。」（場所）
  • 「彼らは...していると思います。」+ 現在分詞（食べている、話している、遊んでいる）
  • 「左側/右側の人は...を着ています。」（服）
  • 「...のようです。」（推測：彼らは幸せそう、天気が良い）

タスク3 — シミュレーション対話（2-3分）

求められること：試験官との実際のコミュニケーション状況をシミュレーションします：店で何かを買う、テーブルを予約する、観光案内所で情報を尋ねる、問題を訴える。

よくある状況：

  • 衣服を購入する：サイズ、価格を尋ねる、試着する、支払う
  • レストラン：メニューを頼む、料理を選ぶ、勘定を頼む
  • ホテル：部屋を予約する、サービスや時間を尋ねる
  • 医者：症状を説明する、指示を理解する
  • 交通：切符を買う、時間を尋ねる、プラットフォーム

キーフレーズ：

  • 「...はいくらですか？」 / 「他のサイズ/色はありますか？」
  • 「...を1つください。」
  • 「...は何時に出発/到着しますか？」
  • 「予約したいのですが...」 / 「可能ですか？」

タスク4 — 自由な会話（2-3分）

求められること：試験官があなたの生活、好み、経験について質問します。自然な会話です。

典型的な質問：

  • 「先週末は何をしましたか？」
  • 「休暇の計画は何ですか？」
  • 「自由な時間に何をするのが好きですか？」
  • 「あなたの普通の日はどのようですか？」
  • 「都市と田舎、どちらが好きですか？なぜですか？」

12分の準備時間

試験官の前に入る前に、12分と紙が与えられ、タスク1の準備をします。次のように活用してください：

  • 1-2分：2つの選択肢を読みます。最も簡単な方を選びます（最も興味深い方ではなく、最も簡単な方）。
  • 3-8分：言いたい4-5点の概要を書きます。完全な文を書く必要はありません（読む時間がありません）。
  • 9-12分：声に出して練習します。モノローグを一度完全に言って、時間を計ります。

試験官が評価すること

  • 流暢さ：過度の間を置かずに話すこと（速さは必要ありませんが、連続性が必要です）
  • 発音：明確に理解できること（完璧なアクセントは必要ありません）
  • 文法：基本的な現在、過去、未来を正しく使うこと
  • 語彙：テーマに適した多様で適切な単語を使うこと
  • インタラクション：質問に答え、会話を続けること

避けるべきエラー

  • 1語の回答：「料理は好きですか？」 → 「はい」✗。より良いのは：「はい、料理がとても好きです。週末にはパスタを作り、デザートも作るのが好きです。」✓
  • 長い沈黙：単語が思い出せない場合は、別の方法で説明します。「その単語は思い出せませんが、薬を買う場所です」（薬局）。
  • 話すのが速すぎる：緊張すると早口になります。ゆっくり、はっきり話してください。
  • メモを1語ずつ読む：概要はガイドであり、読むためのテキストではありません。

自宅での練習方法

  • 毎日異なるテーマについて2分間話している自分を録音する
  • 録音を聞いて、エラーをメモする
  • 写真を説明する練習をする（Googleの画像を使用）
  • 友達や鏡の前で対話をシミュレーションする
  • 迅速な回答を練習するために音声プロンプトのアプリを使用する

インタラクティブな練習問題と会話のプロンプトでDELE A2の口頭表現を練習しましょう。
